Utilisation du dépôt pour la première fois : 

1) Crée un répertoire sur ton à l'aide de ton gestionnaire de fichiers / terminal

2) Ouvre un terminal, accède à ton répetoire : 
cd /home/prénom/.../Web 
(Web est le nom du répertoire)

3) cloner le fichiers existants sur le dépôt dans le répertoire : 
git clone https://github.com/grainedesesame/WEB-groupe6.git


Commandes usuelles : 

* Quand tu crées un nouveau fichier : git add <nom du fichier>
* Quand tu enregistres tes modifications : git commit -am "Message descriptif"
* Quand tu envoies tes enregistrements en ligne : git push
* Quand tu récupères les modifications en ligne : git pull

tout ça depuis le répertoire
